# MR NIRMAN Infra

Advanced responsive website for **MR NIRMAN** — construction, urban service apartments, green living, and future technology in **Hyderabad** & **Bangalore**.

**Tagline:** Building Trust, Creating Futures

## Run locally

```bash
npm install
npm run dev
```

## SEO

- Meta tags (title, description, keywords, robots)
- Open Graph & Twitter cards
- JSON-LD: Organization, LocalBusiness, WebSite, FAQPage, BreadcrumbList
- `public/robots.txt` & `public/sitemap.xml`
- Semantic HTML, skip link, FAQ `<details>`, image alt text
- Set `VITE_SITE_URL` in `.env` when using a custom domain

## Deploy

Push to `main` → GitHub Actions deploys to Pages.

**Live:** https://atshybrid.github.io/nirman/

Submit sitemap in [Google Search Console](https://search.google.com/search-console): `https://atshybrid.github.io/nirman/sitemap.xml`
